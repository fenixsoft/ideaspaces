## 快速开始

### 互联网访问

直接浏览本站，阅读公开文章。

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/fenixsoft/ideaspaces.git

# 安装依赖
npm install

# 启动开发服务器
npm run local
```
