# SVM进阶——优化与扩展

## 引言：从理论到实践

前两章我们学习了SVM的最大间隔原理和核技巧，但还有一些实践问题需要解决：

1. **大规模优化**：当样本量很大时，如何高效求解？
2. **回归问题**：SVM如何处理连续值预测？
3. **多分类问题**：SVM本质是二分类器，如何扩展到多类？

本章将介绍这些进阶内容，并讨论SVM在深度学习时代的价值。

## SMO算法：分解优化

### 对偶问题的挑战

SVM的对偶问题：

$$\max_\alpha \sum_{i=1}^{n} \alpha_i - \frac{1}{2} \sum_{i=1}^{n} \sum_{j=1}^{n} \alpha_i \alpha_j y_i y_j K(x_i, x_j)$$

$$\text{s.t.} \quad 0 \leq \alpha_i \leq C, \quad \sum_{i=1}^{n} \alpha_i y_i = 0$$

当 $n$ 很大时，标准优化方法（如内点法）计算成本高。**SMO（Sequential Minimal Optimization）**是一种高效的分解算法。

### SMO核心思想

**每次只优化两个变量**，固定其他变量不变。

为什么选两个？因为有约束 $\sum_i \alpha_i y_i = 0$，如果只改变一个 $\alpha_i$，约束会被破坏。选择两个变量可以同时满足约束和边界条件。

### SMO算法步骤

1. **选择变量对**：选择 $\alpha_i$ 和 $\alpha_j$ 进行优化
   - 启发式选择：选择违反KKT条件最严重的样本

2. **解析求解**：固定其他变量，对选中的两个变量求解析解
   - 问题转化为单变量二次优化，有闭式解

3. **更新阈值**：更新 $b$ 使KKT条件满足

4. **检查收敛**：检查所有变量是否满足KKT条件

### SMO的优势

- 每次迭代只需计算少量核函数值
- 解析求解无需数值优化
- 内存效率高，适合大规模问题
- 稀疏性：多数 $\alpha_i = 0$，只需存储支持向量

## SVR：支持向量回归

### 问题设定

SVM用于分类，**支持向量回归（SVR）**用于回归预测。

给定数据 $\{(x_i, y_i)\}$，SVR寻找函数 $f(x) = w^T x + b$，使得预测值与真实值的偏差不超过 $\epsilon$。

### ε-不敏感损失

SVR使用**ε-不敏感损失函数**：

$$L_\epsilon(y, f(x)) = \max(0, |y - f(x)| - \epsilon)$$

- 预测误差在 $\epsilon$ 范围内：损失为0
- 预测误差超过 $\epsilon$：损失为超出部分

```
损失
  │    /
  │   /
  │  /
──┼──┼──────┼──┼── 预测值
  │  │      │  │
  └──┴──────┴──┴── 真实值
     -ε     +ε
     
  ε范围内不惩罚
```

### SVR优化问题

$$\min_{w, b} \frac{1}{2} ||w||^2 + C \sum_{i=1}^{n} (\xi_i^+ + \xi_i^-)$$

$$\text{s.t.} \quad y_i - w^T x_i - b \leq \epsilon + \xi_i^+$$
$$\quad\quad w^T x_i + b - y_i \leq \epsilon + \xi_i^-$$
$$\quad\quad \xi_i^+, \xi_i^- \geq 0$$

与分类SVM类似，SVR也有对偶形式和核化版本。

### 与传统回归对比

| 方法 | 损失函数 | 特点 |
|------|----------|------|
| 线性回归 | 平方损失 | 对异常值敏感 |
| SVR | ε-不敏感损失 | 对小误差不敏感，鲁棒 |
| 岭回归 | 平方损失+L2正则 | 稳定但不够稀疏 |

## 多分类扩展

### 一对多（One-vs-Rest, OvR）

训练 $K$ 个二分类器，第 $k$ 个分类器区分"类别 $k$"与"其他所有类别"。

预测时，选择决策函数值最大的类别：

$$\hat{y} = \arg\max_k f_k(x)$$

**优点**：训练 $K$ 个分类器，计算量适中
**缺点**：类别不平衡问题（每个分类器的正负样本比例可能悬殊）

### 一对一（One-vs-One, OvO）

训练 $\frac{K(K-1)}{2}$ 个二分类器，每对类别训练一个分类器。

预测时，使用**投票法**：每个分类器投一票，得票最多的类别胜出。

**优点**：每个分类器数据平衡
**缺点**：分类器数量多，$K$ 大时计算成本高

### 比较

| 方法 | 分类器数量 | 适用场景 |
|------|-----------|----------|
| OvR | $K$ | 类别数多 |
| OvO | $K(K-1)/2$ | 类别数少、数据平衡重要 |

## 与深度学习对比

### SVM的独特价值

尽管深度学习在许多领域超越SVM，但SVM仍有其独特价值：

**1. 小样本学习**

SVM在小样本场景表现稳定，深度学习需要大量数据。

```
样本数量    SVM准确率    深度学习准确率
10          85%          60%
100         92%          85%
1000        94%          93%
10000       95%          97%
```

**2. 理论保证**

SVM基于统计学习理论，有**泛化误差界**的理论保证：

$$R(f) \leq R_{emp}(f) + \sqrt{\frac{h(\log(2n/h) + 1) - \log(\delta/4)}{n}}$$

其中 $h$ 是VC维，$R_{emp}$ 是经验风险。这提供了模型选择的依据。

**3. 可解释性**

SVM的决策函数清晰：

$$f(x) = \sum_{i \in SV} \alpha_i y_i K(x_i, x) + b$$

支持向量明确告诉我们"哪些样本决定了边界"。

**4. 高维稀疏数据**

文本分类等场景，特征维度远大于样本数，SVM表现优异。核方法在高维空间自然有效。

**5. 计算效率**

训练好的SVM只需要存储支持向量，预测速度快。对于实时预测场景有优势。

### SVM的局限

当然，SVM也有明显局限：

1. **大数据困难**：训练复杂度 $O(n^2)$ 到 $O(n^3)$，难以处理百万级样本
2. **特征自动学习**：无法像深度学习那样自动学习特征表示
3. **多模态数据**：图像、语音等复杂数据，深度学习效果更好
4. **调参复杂**：核函数选择、参数调节需要经验

### SVM与深度学习的结合

现代应用中，SVM常与深度学习结合：

- **深度特征 + SVM分类器**：用CNN提取特征，用SVM做最终分类
- **迁移学习**：预训练模型提取特征，SVM做领域适应
- **集成方法**：SVM作为集成学习的一个基学习器

---

## 小结

本章介绍了SVM的进阶内容：

1. **SMO算法**：分解优化，每次优化两个变量，高效处理大规模问题
2. **SVR回归**：使用ε-不敏感损失，对预测误差在一定范围内不敏感
3. **多分类扩展**：一对多（OvR）和一对一（OvO）两种策略
4. **深度学习对比**：SVM在小样本、理论保证、可解释性方面有独特价值

SVM是统计学习的经典代表。它将分类问题转化为凸优化问题，有坚实的理论基础。虽然深度学习在很多领域取代了SVM，但理解SVM有助于建立对机器学习的完整认知——从几何直觉到优化理论，从核方法到泛化保证。这些思想在现代机器学习中依然重要。
